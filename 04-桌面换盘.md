## 问题说明

很多时候C盘空间不够，其中原因就在于很多桌面文件属于C盘，本文介绍如何把桌面换个盘。

---

## 操作步骤

### 步骤1：打开运行对话框
- 点击开始菜单，选择"运行"工具
- 或按下快捷键 **Win+R** 打开运行菜单

### 步骤2：打开注册表编辑器
- 在"运行"对话框中输入 `regedit`
- 按下确定，打开注册表编辑器

### 步骤3：定位到目标路径

在注册列表中依次展开以下路径：

```
HKEY_CURRENT_USER → Software → Microsoft → Windows → CurrentVersion → Explorer → User Shell Folders
```

> 快速复制路径：
> `\HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders`

### 步骤4：找到桌面注册项
- 在注册表窗口右边找到 **"Desktop"**
- 如果没修改过系统，默认值为用户配置文件路径

### 步骤5：修改桌面路径
- 双击 **Desktop**
- 在弹出的对话框中，修改其数据为预设好的路径
- **示例**：想把桌面放在"D盘-桌面"文件夹，修改值为 `D:\桌面`
- 路径名称可以自定义，中英文均可

---

## 核心要点

| 要点 | 说明 |
|------|------|
| 问题根源 | 桌面文件默认存储在C盘，占用系统盘空间 |
| 解决方案 | 通过修改Windows注册表，将桌面路径指向其他盘符 |
| 注意事项 | 修改前建议先在目标位置创建好文件夹；修改后原桌面文件不会自动迁移 |
