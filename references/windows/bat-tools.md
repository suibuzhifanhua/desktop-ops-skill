# Windows 实用 BAT 工具合集

> 以下脚本均来自实战积累，按需取用。
> **使用方式**：新建记事本 → 粘贴对应代码 → 另存为 `.bat` 文件（编码选 ANSI）→ 双击运行。
> 部分脚本需要**管理员权限**运行（右键 → 以管理员身份运行）。

---

## 🔑 查询序列号 & 网络信息

**文件名建议**：`序列号.bat`

```bat
@echo off
wmic path softwarelicensingservice get OA3xOriginalProductKey

@echo off
ipconfig /all

pause
```

**用途说明**：
- 第一条命令：查询 Windows 系统的 **OEM 序列号**（激活密钥），适用于重装系统时找回原厂密钥；
- 第二条命令：输出完整网络信息，包含 IP 地址、MAC 地址、DNS 服务器等；
- `pause` 让窗口停住不关闭，方便查看结果。

> 💡 双击直接运行即可，不需要管理员权限。

## 🗑️ 强制删除文件或文件夹

**文件名建议**：`强制删除.bat`

```bat
del /f /s /q %1
rd /s /q %1
```

**用途说明**：
- 解决"文件正被占用无法删除"或"路径过长无法删除"等问题；
- `del /f /s /q`：强制删除文件夹内所有文件（忽略只读属性、不弹确认框）；
- `rd /s /q`：删除空文件夹本身；
- `%1`：占位符，代表拖入的目标路径。

**使用方式**：
1. 保存好 `.bat` 文件后，将**想要删除的文件或文件夹直接拖拽到此 bat 文件上**；
2. 脚本自动执行强制删除，无弹窗确认。

> ⚠️ 此操作**不可撤销**，请确认目标无误后再拖入。

## ⬆️ 一键更新所有软件

**文件名建议**：`更新.bat`

```bat
winget upgrade --all --include-unknown --accept-package-agreements
pause
```

**用途说明**：
- 使用 Windows 内置的 `winget` 包管理器，**一次性更新电脑上所有可识别的软件**；
- `--include-unknown`：将版本未知的软件也纳入更新范围；
- `--accept-package-agreements`：自动接受所有软件许可协议，无需手动确认；
- `pause`：更新完成后暂停窗口，方便查看更新日志。

**前提条件**：
- Win10 1709 及以上版本自带 `winget`；若提示找不到命令，请先到微软商店安装「应用安装程序」；
- 需要联网，更新内容越多耗时越长（通常 5~30 分钟）。

> 💡 建议每周或每月运行一次，保持软件版本最新。

## 🧹 清理临时文件缓存

**文件名建议**：`清理缓存.bat`

```bat
@echo off
rd /s /q %temp%
echo Temp文件夹已清空
pause
```

**用途说明**：
- 清除 Windows 用户级临时文件目录（`%temp%`，通常为 `C:\Users\用户名\AppData\Local\Temp`）；
- `rd /s /q`：递归删除目录下所有内容，不提示确认；
- 释放 C 盘空间，可消除部分软件崩溃或安装失败时遗留的垃圾文件。

**使用方式**：
1. **右键 → 以管理员身份运行**（普通权限有时无法删除被系统占用的临时文件）；
2. 运行后会打印"Temp文件夹已清空"，然后暂停等待按键退出。

> 💡 正在运行的程序产生的临时文件可能删不掉（提示访问被拒绝），属正常现象，关掉对应程序后重试即可。

## 🔒 快速锁屏

**文件名建议**：`锁屏.bat`

```bat
rundll32.exe user32.dll LockWorkStation
```

**用途说明**：
- 调用 Windows API 立即锁定工作站，效果等同于 `Win + L`；
- 离开工位时双击一下即可锁屏，比找快捷键更方便；
- 可以把此文件放在桌面，或固定到任务栏快速访问。

> 💡 也可以给此 bat 文件创建桌面快捷方式，右键 → 属性 → 更改图标，换成锁的图标，更直观。
