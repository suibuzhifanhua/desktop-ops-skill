## 🎮 游戏性能优化

### 关闭 VBS / HVCI 提升游戏帧率（提升 10-15%）

**方法一：命令行**
```powershell
# 以管理员身份打开终端，执行：
bcdedit /set hypervisorlaunchtype off
# 重启电脑生效
```

**方法二：图形界面**
```
设置 → 隐私和安全性 → Windows 安全中心 → 设备安全性 → 内存完整性 → 关闭
```

> ⚠️ 注意：关闭此安全功能可能降低系统防护，游戏玩家可根据需求权衡。

### 启用高性能电源模式

**路径一**：`设置 → 系统 → 电源和电池` → 电源模式选择「最佳性能」

**路径二**：控制面板搜索「电源计划」→ 选择「高性能」

> 部分品牌机需安装厂商电源管理软件才能切换。

## ⚡ 系统响应优化

### 关闭视觉动画效果（老旧电脑提速明显）

```
此电脑 → 右键属性 → 高级系统设置 → 性能设置 → 调整为最佳性能
```

或手动取消勾选以下项（保留「平滑屏幕字体边缘」）：
- ❌ 窗口内的动画控件和元素
- ❌ 淡入淡出或滑动菜单到视图
- ❌ 在点击后淡出菜单
- ❌ 在视图中淡入

### 禁用冗余系统服务

使用任务管理器 → 启动选项卡，禁用非必要启动项：
- Adobe 自动更新
- 品牌机冗余硬件助手
- 不常用的云同步服务

> 建议保留：杀毒软件、显卡驱动、音效管理

### 关闭快速启动（解决关机/开机异常）

```
控制面板 → 电源选项 → 选择电源按钮的功能 → 关机设置 → 取消勾选「启用快速启动」
```

## 💾 磁盘空间优化

### 开启存储感知（自动清理）

```
设置 → 系统 → 存储 → 启用存储感知
```

设置每周自动清理临时文件、回收站过期内容。

### 清理 Windows.old（旧系统备份，节省 5-15GB）

```
此电脑 → 右键 C 盘 → 属性 → 磁盘清理 → 清理系统文件 → 全部勾选 → 确定
```

> ⚠️ 建议在系统更新后 10 天内完成清理，否则 Windows.old 会被自动删除。

### 转移默认存储路径（C 盘搬家）

```
设置 → 系统 → 存储 → 高级存储设置 → 保存新内容的地方
```

可将「文档」「下载」「图片」等默认路径从 C 盘改至其他分区。

## 🧠 内存优化

### 使用 RAMMap 分析内存占用

工具下载：[RAMMap](https://learn.microsoft.com/en-us/sysinternals/downloads/rammap)（Sysinternals 官方）

**常用操作**：
| 操作 | 效果 |
|------|------|
| Empty Working Sets | 清空进程工作集，释放被占用但空闲的内存 |
| Empty Standby List | 清空待机列表，释放缓存占用的内存 |

> 配合任务管理器观察清理效果。

### 解决内存泄漏导致卡顿

1. 任务管理器 → 结束高内存进程
2. 禁用 SysMain + PrintSpooler 服务：`services.msc` → 找到服务 → 禁用
3. 关闭快速启动（见上方）

## 🔒 隐私优化

### 关闭系统广告

```
设置 → 隐私和安全性 → 常规
```

关闭所有「个性化」「广告追踪」相关选项。

### 控制系统更新

**工具法**：使用 [Windows Update Blocker](https://www.sordum.org/9479/windows-update-blocker/) 一键禁用更新服务。

> ⚠️ 警告：长期关闭更新可能带来安全风险，建议仅在必要时临时使用。

## IE 自动跳转 Edge 的处理

> 来源：[Fisheep的新世界](https://fisheep.fun/yummy/90)

Windows 10/11 中，IE 浏览器会自动跳转到 Edge，若需阻止此行为：

1. 打开 **Edge 浏览器**
2. 点击右上角 **"…"**（更多操作）→ **设置**
3. 左侧选择 **"默认浏览器"**
4. 找到 **"让 Internet Explorer 在 Microsoft Edge 中打开站点"**
5. 将其从 **"始终"** 改为 **"不兼容的站点"** 或 **"从不"**

> ⚠️ 注意：微软已逐步淘汰 IE，建议优先使用 Edge 的 IE 兼容模式访问老旧网站。
