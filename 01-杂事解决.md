## 1. 电脑开机无反应，点电源键无作用怎么办？

**解决方案**：
- 首先检查各电源是否插好通电
- 如果不行，拆开机器将机器中的纽扣电池拔出来，等待一两分钟再装回去
- 年代久远的机器建议更换一个新的纽扣电池

---

## 2. 一直停留在BIOS界面，按键盘无反应怎么办？

**解决方案**：
- 首先排除键盘问题，换一个键盘试试
- 如果不行，就是硬盘出问题了
- 查看屏幕显示的BIOS界面提示信息
- 如果提示硬盘问题，按F1可以正常进入系统
- 建议马上排查硬盘问题，因为可能会危害数据

## 6. 大部分Win10自动更新后C盘空间变少怎么办？

**解决方案**：
1. 在桌面找到"此电脑"
2. 找到C盘，右键点击"属性"
3. 点击"磁盘清理"
4. 点击"清理系统文件"
5. 全部勾选
6. 点击确定

## 11. Win10卓越性能模式怎么打开？

**解决方案**：
> 主要针对部分笔记本机型

1. 右键Win图标
2. 选择PowerShell管理员模式
3. 输入以下命令：

```cmd
powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
```

## 14. 没有DiskGenius等软件，如何查看电脑磁盘是HDD还是SSD？

**解决方案**：
1. 打开PowerShell（管理员模式）
2. 输入命令：

```powershell
get-physicaldisk
```

3. 查看 **MediaType** 即可知道磁盘类型

## 16. 扫描修复Windows

**解决方案**：
1. 按 **Windows徽标键+X**，启动"Windows PowerShell（管理员）"
2. 输入以下命令：

```cmd
Dism /Online /Cleanup-Image /ScanHealth
```
*此命令将扫描全部系统文件并和官方系统文件对比*

3. 检查系统文件是否有损坏：

```cmd
Dism /Online /Cleanup-Image /CheckHealth
```
*此命令必须在前一条命令执行完以后，发现系统文件有损坏时使用*

4. 还原系统文件：

```cmd
DISM /Online /Cleanup-image /RestoreHealth
```
*此命令把不同的系统文件还原成官方系统源文件*

5. 完成后重启，再键入：

```cmd
sfc /SCANNOW
```

## 21. 查本机CPU/内存条

**解决方案**：

**方法一：命令行**
1. Win+R打开运行，输入cmd回车
2. 在命令行界面输入 `wmic` 进入系统管理执行脚本界面
3. 通过命令 `memorychip` 查看内存型号（可看出内存条数量和PN号）
4. 通过命令 `cpu get *` 查看CPU具体情况
   - 找到 `numberofcores`（核心数）和 `numberoflogicalprocessors`（逻辑处理器数）

**方法二**：下载鲁大师

## 22. 电脑突然关机断电

**解决方案**：
1. 首先排除停电因素
2. 打开机箱
3. 把纽扣电池取出来摩擦摩擦再放回去
4. 盖好后盖就可以开机了

---

## 23. 挂载的映射盘重启后或一段时间后断开怎么办？

**解决方案**：
1. 右键开始
2. 打开管理员模式下的PowerShell
3. 输入代码：

```cmd
net config server /autodisconnect:-1
```

## 24. 怎么关闭休眠模式（减少C盘占用，但开机慢一点）

**解决方案**：
- 管理员打开CMD
- 输入：

```cmd
powercfg -h off
```

## 26. 无法删除文件夹

**解决方案**：
1. 新建一个bat文件，内容如下：

```batch
DEL /F /A /Q \\?\%1
RD /S /Q \\?\%1
```

2. 将需要删除的文件夹拖到bat文件上即可

## 27. Aliyun删除不必要进程

**解决方案**（Linux服务器）：
```bash
wget http://update.aegis.aliyun.com/download/uninstall.sh
chmod +x uninstall.sh
./uninstall.sh

wget http://update.aegis.aliyun.com/download/quartz_uninstall.sh
chmod +x quartz_uninstall.sh
./quartz_uninstall.sh

pkill aliyun-service
rm -fr /etc/init.d/agentwatch /usr/sbin/aliyun-service
rm -rf /usr/local/aegis*
```
